Does it need WiFi?

Lite (49KB): No. Connects via BLE or serial to a nearby gateway (phone, Pi, laptop). The gateway handles LLM API calls.
Full (‚â§500KB): Yes. Connects directly to LLM APIs over WiFi/HTTP.
